# Get Started

## Install Aral

You can download the `aral.exe` file from the [Releases](https://github.com/erinshek/aral-lang/releases/tag/v0.1.0) section and use it to run and test code written in the **Aral programming language**.

## Quickstart

The code writing syntax of the Aral programming language is as simple as Python's. 
You can see this in the example of the "Hello World" code below.

```aral
shıǵar("Hello World");
```

## Run

```shell
.\aral.exe run program.aral
```

## Syntax

The syntax of the Aral programming language is in Karakalpak. 
All keywords and commands are written using words from the Karakalpak language.
